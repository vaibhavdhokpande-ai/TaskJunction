Three-Phase Phasor Diagram
How to read: VR (red) is the reference phasor at 0°. VY (yellow) lags by 120° and VB (blue) lags by 240°. The current phasors (dashed) show the phase angle φ relative to each voltage. Run the calculator first to update angles.

⚡ Phase Sequence

  • Standard: R → Y → B (positive)
  • Reverse: R → B → Y (negative)
  • India: R-Y-B per IE Rules
  • IEC: L1-L2-L3 (same)
  • Verify with phase sequence meter

📐 Voltage Relationships

  • Star: VLL = √3 × VLN
  • 415 V LL → 239.6 V LN
  • 11 kV LL → 6.35 kV LN
  • Delta: Vphase = Vline
  • VLN = VLL / 1.732

🔁 Current Relationships

  • Star: IL = Iphase
  • Delta: IL = √3 × Iphase
  • Neutral current = vector sum
  • Balanced: IN = 0
  • Unbalanced: IN ≠ 0 (measure)

📊 DISCOM PF Penalty Guidelines (India)

  • PF < 0.85 → Penalty surcharge (typically 0.5–1% per 0.01 drop)
  • PF > 0.95 → Incentive / rebate in many DISCOMs
  • IE Rules 1956 Rule 51: minimum PF 0.85
  • CERC regulation: PF ≥ 0.90 recommended
  • Industrial HT consumers: metered kVAh billing
  • Always check local DISCOM tariff schedule

🔋 Capacitor Bank Sizing Tips

  • Install at point of use (motor terminals) for best results
  • Avoid over-correction — leading PF can cause voltage rise
  • For variable loads: use automatic APFC panels
  • Standard steps: 5, 10, 15, 20, 25, 30, 50 kVAR
  • Derating: 0.9 × nameplate for 50°C ambient
  • Verify harmonic levels before installing capacitors
Three-Phase Power Formulas
Balanced Three-Phase (Star or Delta)
P = √3 × VLL × IL × cos φ = 3 × VLN × IL × cos φ
Q = √3 × VLL × IL × sin φ = 3 × VLN × IL × sin φ
S = √3 × VLL × IL = √(P² + Q²)
PF = cos φ = P / S

Unbalanced Three-Phase
Ptotal = PR + PY + PB = VRIRcosφR + VYIYcosφY + VBIBcosφB
Qtotal = QR + QY + QB = VRIRsinφR + VYIYsinφY + VBIBsinφB
Stotal = √(P² + Q²)

Star ↔ Delta Conversion
VLN = VLL / √3 = VLL / 1.7321
IΔ = IL / √3 (Delta phase current)
PF Correction: QC = P × (tan φ1 − tan φ2) [kVAR]

🏭 Typical Load Power Factors

  • Induction motors (full load): 0.85–0.92
  • Induction motors (light load): 0.40–0.65
  • Welding machines: 0.35–0.60
  • Fluorescent lighting: 0.85–0.95
  • LED lighting: 0.90–0.99
  • Resistive heaters: 1.00
  • Variable speed drives: 0.95–0.98
  • UPS systems: 0.80–0.90 (input)
