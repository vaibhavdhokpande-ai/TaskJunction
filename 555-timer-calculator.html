555 Timer Calculator

Design astable and monostable 555 timer circuits. Compute frequency, duty cycle, pulse width and component values.

🔄 Astable Mode — Free-Running Oscillator
Generates a continuous square wave. Configure R1, R2 and C to set frequency and duty cycle.
Mode: Select how to design the circuit — either by entering R, C values, or by specifying the target frequency/duty cycle.

Note: Astable duty cycle > 50% (standard config). For < 50%, add a diode across R2.

📊 Output Waveform Preview
Charge time (t_high): 0.693 × (R1 + R2) × C
Discharge time (t_low): 0.693 × R2 × C
Period: T = t_high + t_low
Frequency: f = 1/T = 1.44 / [(R1 + 2×R2) × C]
Duty Cycle: D = (R1 + R2) / (R1 + 2×R2) × 100%
⏱️ Monostable Mode — One-Shot Timer
A single trigger pulse generates one output pulse of defined width. T = 1.1 × R × C
Pulse Width: t = 1.1 × R × C
Threshold: Vout goes HIGH when trigger < Vcc/3
Output: Single pulse, then returns LOW
To design: R = t / (1.1 × C)
