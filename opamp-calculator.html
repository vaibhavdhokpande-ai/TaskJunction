Op-Amp Circuit Calculator

Design inverting, non-inverting, summing and difference amplifiers with gain, bandwidth and output swing analysis.

➖ Inverting Amplifier
Output is inverted (180° phase shift). Gain = −Rf / Rin
Gain: Av = −Rf / Rin
Bandwidth: BW = GBW / |Av|
Input impedance: Zin = Rin
Vout: clipped to ±(Vs − 1.5V) typically
➕ Non-Inverting Amplifier
Output is in phase with input. Gain = 1 + Rf/R1 (minimum gain = 1)
Gain: Av = 1 + Rf / R1
Bandwidth: BW = GBW / Av
Input impedance: Very high (ideally infinite)
Unity Gain Buffer: Short Rf = 0, remove R1 → Av = 1
∑ Summing (Mixer) Amplifier
Inverts and sums weighted inputs. Vout = −Rf × (V1/R1 + V2/R2 + V3/R3)
Vout = −Rf × (V1/R1 + V2/R2 + … + Vn/Rn)
For equal weights: all Rin = Rf → Vout = −(V1+V2+…+Vn)
△ Difference Amplifier
Amplifies the difference between two signals. Vout = (Rf/R1)(V2 − V1)
Gain: Av = R2 / R1 (when R1=R3, R2=Rf)
Vout = (Rf/R1) × (V2 − V1)
CMRR: Common Mode Rejection Ratio in dB
